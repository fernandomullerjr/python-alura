# 03 Saindo do loop

Considere o seguinte trecho de código com um loop:

~~~~PYTHON
passos = 0
while (#######):
  passos += 1
print(passos)
~~~~

O que precisa ser colocado na condição do while para que o loop saia após 10 passos?
Selecione 2 alternativas

    passos<10

    passos==10

    passos<=9

    passos<=10
